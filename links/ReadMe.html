Interactive Web Map of Airports in United States

Description:

The Map shows air traffic control tower data for airports in United States. It consists of state boundaries and airport locations. The choropleth map describes the number of airports within each state.

Functions:

  • Click on state to find the name of the state.
  • Click on tower icon on the map to find the name of the airport.
  • Use pan, and zoom in and out to move quickly to any area of the map.

Credits:

The airport data is obtained from USGS (https://catalog.data.gov/dataset/usgs-small-scale-dataset-airports-of-the-united-states-201207-shapefile). The US state boundaries data is acquired from Mike Bostock of D3. The basemap is provided by CartoDB.
