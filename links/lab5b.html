Assignment 5 - ArcGIS Online Map - Median House Prices in Southeast Michigan

(view larger map)

Description:

The map shows census tract boundaries in seven counties, namely, Livingston, Oakland, Macomb, Monroe, St. Clair, Washtenaw, and Wayne, in southeast Michigan region. The counties are identified in the map using API number. The map contains socio-economic data from the 2010 Census and displays the median house prices per census tracts using heat map.
