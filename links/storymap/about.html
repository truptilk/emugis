Bali and Gili Islands, Indonesia

This is a story map of tourist places in Bali and Gili islands in Indonesia. There are so many to mention but a few famous spots are included in this story map. It can be considered as a spatial tour blog.

The story map is created using reference materials available at https://github.com/jakobzhao/storymap provided by Bo Zhao.

The favicon image is created using favicon.io.

Credits for the background video on first page and image on last page: Trupti Lokhande.

Libraries/Stylesheets used:

  • jQuery
  • Leaflet 1.4
  • Leaflet-ajax
  • Storymap 2.5
  • Bootstrap
  • Google Material Icon
  • Font Awesome

Features included:

  • Scenes with Fullscreen Image and Video
  • Navigation Bar
  • Tile Layers
  • GeoJson Layers
  • Legend
  • Scale Bar
  • Favicon image
  • Social Media Button
  • Info Page Button

The story map is created in partial fulfilment of the requirements for the coursework GEOG 678 Geovisualization: Web Mapping.
